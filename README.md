## Sarah Smallwood Portfolio
This project was created with the React App. Using a GitHub API, I successfully linked some select GitHub Repositories that showcase some of my best work. These projects include Mock-Ups and a resume. 

I have also included a skill list that displays that languages and technologies I have experience with. 
Using a React Import Library, I was able to display some of my favorite travel photos in the travel gallery.

Links in the Navigation Bar show icons to connect to my social media networks.
